# 使用指南

### 安装

```sh
git clone https://github.com/EsunR/Blog-Index.git
cd ./Blog-Index
git checkout dev
yarn install
```

### 开发预览

```sh
yarn dev
```

### 打包编译

```sh
yarn build
```

> 生成的静态文件资源在 dist 目录下
